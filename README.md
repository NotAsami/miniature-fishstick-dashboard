# Miniature Fishstick Dashboard
A simple personal dashboard built with Vite, React, and Tailwind CSS.
This project serves as a beginner-friendly starting point for building a modern, expandable web interface that integrates with various APIs (such as weather, news, or custom data sources).

## Features

* Modern front-end using React
* Utility-first styling with Tailwind CSS
* Organized project structure for scalability
* Easily expandable with additional components and API integrations

## Getting Started

### 1. Install Dependencies

```
npm install
```

### 2. Start Development Server

```
npm run dev
```

### 3. Build for Production

```
npm run build
```

### 4. Preview Production Build

```
npm run preview
```

## Requirements

* Node.js (LTS recommended)
* npm or yarn

## Notes

This project is intended as a simple template. You can extend it with more components and integrate additional APIs over time as your dashboard grows.
